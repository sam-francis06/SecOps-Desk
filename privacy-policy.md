# Privacy Policy — SecOps Desk (Zoho Cliq Extension)

_Last Updated: November 18, 2025_

SecOps Desk is a cybersecurity extension built for Zoho Cliq.  
We respect user privacy and are committed to protecting it.

## 📌 Data We Do Not Collect
SecOps Desk **does not collect, store, or transmit any personal data**.

We do NOT access or store:
- Usernames
- Emails
- Zoho account details
- Chat messages
- Personal identifiers
- Organization data

## 🔐 How the Extension Works
All lookups are:
- Triggered manually by the user
- Handled inside Zoho Cliq
- Not logged or saved externally

Threat lookups (IP, domain, URL, CVE) are made **only by user request**.

## 🌐 Third-Party APIs
The extension connects to:
- VirusTotal
- AbuseIPDB
- WHOISXML
- IP-API

These services return **public cybersecurity intelligence**, not personal data.

## 📦 Data Storage
We do *not* store:
- Logs
- API responses
- User actions
- IP addresses

All data remains within the user’s Zoho workspace.
